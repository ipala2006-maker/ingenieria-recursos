# Estudiemos

Sitio estatico de recursos de estudio para ingenieria.

## Estado actual

- La pagina funciona sin backend.
- No hay automatizacion activa con OpenAI API.
- `TASKS.md` queda como libreta simple para anotar tareas desde el celular.

## Anotar tareas desde el celular

1. Abrir `TASKS.md`.
2. Escribir una tarea debajo de `## Pendientes`.
3. Guardar el cambio.
4. Avisar en Codex cuando quieras que se trabaje esa tarea.

Ejemplo:

```md
- [ ] Revisar cards de materias en mobile.
```

## Notas para futuras automatizaciones

Si mas adelante se quiere volver a usar automatizacion con IA, crearla de nuevo desde cero y revisar costos antes de activarla.
